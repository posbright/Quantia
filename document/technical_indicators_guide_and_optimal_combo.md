## 1. 指标的五大信息源（必看的世界观）

A 股可观察价格信息可分为五类，**任何「胜率提升」都来自跨类别组合，而非同类堆叠**：

| 类别 | 指标举例 | 物理含义 | 在本仓库的实现 |
|---|---|---|---|
| **A. 趋势方向** | MA5/10/20/60/120、SuperTrend、MACD（位置） | 当前是站多、站空，还是震荡 | `quantia/core/indicator/` MACD/EMA |
| **B. 动量速度** | RSI、KDJ、WR、CCI、StochRSI、ROC | 价格变动的速率/超买超卖程度 | RSI6/12/14/24, KDJ-K/D/J |
| **C. 波动率/通道** | Bollinger、ATR、Keltner、ENE、`boll_width` | 极值距离 + 行情类型（震荡 vs 趋势） | BOLL(20,2), ATR14 |
| **D. 资金/量能** | OBV、VR、MFI、`vol_ratio_5`、量比 | 价格背后是否有真金白银 | OBV, VR, MFI, vol_ratio |
| **E. 形态/结构** | DPO、PSY、Pivot、Donchian、形态识别 | 周期、群体心理、关键位 | `cn_stock_kline_pattern` 32 种形态 |

**核心规则**：

- 一个组合**至少跨 3 个类别**；
- 同类别**最多 2 个指标**且互为「同向确认」（不要 RSI + KDJ + WR 三件事变一件事）；
- 「**趋势 + 量能 + 触发事件**」是 A 股最有效的三件套。

## 2. 趋势类指标（A）

### 2.1 移动平均线 MA / EMA

- **算法**：`MA_n = SMA(close, n)` 或 `EMA_n = α·close + (1-α)·EMA_{n-1}`，常用周期 5/10/20/60/120/250。
- **物理含义**：过去 n 日成本线；价格站上意味着多数持仓盈利。
- **默认推荐**：日线 MA20（月线）、MA60（季线）、MA120（半年线）。
- **优势**：直观、稳定、和市场认知高度一致 → 多数交易员的实际成本参考。
- **致命缺陷**：**完全是滞后指标**。震荡市中频繁假突破。
- **何时不用**：周期 < 5 日的短线、单日博弈。
- **实战要点**：单条 MA 价值低；**MA 多头排列（短>中>长）+ MA20 不破** 才是有效信号。

### 2.2 MACD

- **算法**：`DIF = EMA12 - EMA26`，`DEA = EMA9(DIF)`，`柱 = (DIF - DEA) × 2`。
- **物理含义**：短中期动量差，柱子代表趋势的「加速度」。
- **优势**：金叉/死叉、零轴上下、顶/底背离三类信号都很经典；柱子转向比 DIF 穿越早 1–3 日。
- **致命缺陷**：**所有 MACD 信号在震荡市的虚假率 50%+**；零轴附近反复死金叉是真正的杀手。
- **何时不用**：盘整周期；ADX < 18 时直接关闭 MACD 信号。
- **实战要点**：
  - **不要单独用 MACD 决策**，必须叠加 ADX > 22 或趋势确认（MA 多头排列）。
  - 「**MACD 柱由负转正 + 量比 > 1.5**」是高质量买点（V2/V3 报告反复验证）。

### 2.3 SuperTrend

- **算法**：`upper_band = (H+L)/2 + mult·ATR`，`lower_band = (H+L)/2 - mult·ATR`，价格穿越翻向。
- **物理含义**：自适应趋势线 + 动态止损位。
- **优势**：直接给出「多/空」二元状态，可作为复合指标的「方向开关」（V3 中 E1/E3 都用它）。
- **致命缺陷**：参数（period=10, mult=3）在不同股票上差别巨大；高波动股容易频繁翻转。
- **何时不用**：周期 < 5 日；股票流动性差或日内振幅 > 5%。

### 2.4 ADX / DMI

- **算法**：DMI 差值的平均；范围 0–100。
- **物理含义**：**趋势强度**（不是方向）；ADX > 25 = 强趋势，< 18 = 震荡。
- **优势**：是**给所有趋势策略加门槛的最重要指标**；过滤震荡市的虚假突破。
- **致命缺陷**：滞后；ADX 高位钝化（持续 50+ 一段时间后趋势已近末端）。
- **实战要点**：把 ADX > 22 作为 E3 / 趋势策略的硬门槛，胜率立刻提升 5–10%。

### 2.5 TRIX

- **算法**：close 三次 EMA 平滑后的变化率。
- **物理含义**：**长周期** MACD（更平滑、更滞后）。
- **优势**：周线/月线判断大趋势效果好。
- **致命缺陷**：日线短线策略基本无用。
- **何时不用**：周期 < 20 日。

## 3. 动量类指标（B）

### 3.1 RSI（相对强弱指数）

- **算法**：`RSI = 100 - 100/(1 + 平均涨幅/平均跌幅)`，常用周期 6/12/14/24。
- **物理含义**：N 日内涨势相对强度，0–100。
- **默认参数**：14（最经典）；A 股短线偏好 6 或 9。
- **优势**：直观；** 30/70 阈值与 50 中轴线** 都有意义。
- **致命缺陷**：
  - 在强趋势中 RSI 长期 > 70 或 < 30，「超买超卖」失效；
  - 周期短（RSI6）抖动大，周期长（RSI24）反应慢。
- **何时不用**：单边趋势的极端市场（牛市顶部 / 熊市底部）。
- **实战要点**：
  - 「**RSI 突破 50**」是「动量方向切换」的早期信号；
  - 「**RSI 顶背离**」（价格新高、RSI 不创新高）是经典见顶信号；
  - **作为复合指标的动量分量比单用强 10x**。

### 3.2 KDJ

- **算法**：`RSV = (close - LL_n)/(HH_n - LL_n) × 100`，K=EMA(RSV)、D=EMA(K)、J=3K-2D。
- **物理含义**：基于「N 日极值范围」的动量速率。
- **优势**：J 线敏感度高（可超出 0–100）；金叉/死叉信号视觉清晰。
- **致命缺陷**：
  - 在 A 股震荡市频繁金叉死叉，**短线虚假率 50%+**；
  - K/D 与 RSI 相关性 0.85+，**叠加 RSI 几乎没增量信息**。
- **何时不用**：和 RSI 一起作为「动量信息源」的双重保险。
- **实战要点**：「**J 上穿 0**」+ 「价格已企稳」 = 强反弹信号。

### 3.3 WR（威廉指标）

- **算法**：`WR = -100 × (HH_n - close)/(HH_n - LL_n)`，值域 -100 ~ 0。
- **物理含义**：与 KDJ 同源但**值域反向**（WR 越接近 0 = 越超买）。
- **优势**：参数简单；易反转。
- **致命缺陷**：值域反向是常见误用源；与 RSI/KDJ 相关性 0.85+，**单独叠加无效**。
- **实战要点**：归一化时必须用 `100 + WR` 反向修正再相加。

### 3.4 CCI（商品通道指数）

- **算法**：典型价偏离均值的程度。
- **物理含义**：±100 是常态边界，±200 是极端。
- **优势**：在「**横盘振荡市**」表现优于 RSI（因为 CCI 不被 0/100 锁死）。
- **致命缺陷**：值域开放，跨股票阈值不可比。
- **实战要点**：归一化后作为复合指标分量，不要直接做信号。

### 3.5 ROC / Momentum

- **算法**：`ROC_n = (close - close.shift(n)) / close.shift(n)`。
- **物理含义**：纯粹的 N 日涨跌幅。
- **优势**：最简单、零滞后。
- **致命缺陷**：噪声极大；几乎无法独立成为信号。
- **何时使用**：作为复合指标的「速度因子」。

## 4. 波动率/通道类指标（C）

### 4.1 布林带 BOLL

- **算法**：`mid = MA20`，`upper/lower = mid ± 2·std20`。
- **物理含义**：价格 95.4% 时间应在 ±2σ 内。
- **优势**：
  - **三大信号**：穿越下轨（超卖）、`%B = (close-lower)/(upper-lower)`、缩口（`width = (upper-lower)/mid`）；
  - **缩口** 是预测大波动的最佳形态指标之一。
- **致命缺陷**：在单边趋势中价格沿着上轨/下轨持续运行，「均值回归」假设失效。
- **实战要点**：
  - **`boll_width < median60(boll_width) × 0.85`** = 波动率收敛 → 即将爆发；
  - **`%B < 5`** + **当日阳线** = 高质量底部信号（V3 报告 S12 / E2 的核心）；
  - **价格上穿上轨 + ADX > 22 + 量比 > 1.5** = 突破型买点（V2 E3）。

### 4.2 ATR（平均真实波幅）

- **算法**：`TR = max(H-L, |H-C_prev|, |L-C_prev|)`，`ATR = MA(TR, 14)`。
- **物理含义**：股票当前的「正常波动量」（绝对值）。
- **优势**：**最适合用作止损距离**（如 stop = entry - 2·ATR），跨股票通用。
- **致命缺陷**：本身不构成信号。
- **实战要点**：实盘止损用 `2·ATR` 或 `3·ATR`，比固定 5% 适应性强 5x。

### 4.3 ENE / Keltner Channel

- **算法**：基于 EMA 与 ATR 构造的通道。
- **优势**：在低波动股票上比 BOLL 更稳。
- **致命缺陷**：A 股使用率较低，社区共识不如 BOLL。

## 5. 资金/量能类指标（D）

### 5.1 OBV（能量潮）

- **算法**：阳线累加、阴线累减成交量。
- **物理含义**：资金流向的累计真实情况。
- **优势**：**OBV 与价格背离是顶/底最可靠信号**（V3 证明：OBV 斜率 > 60 日中位数是 E2 的核心 alpha 源）。
- **致命缺陷**：原始 OBV 数值无意义，必须看斜率/差分。
- **实战要点**：用 `OBV.diff(10) / |OBV|.rolling(60).mean()` 做归一化。

### 5.2 VR（成交量比率）

- **算法**：N 日内涨日成交量总和 ÷ 跌日成交量总和。
- **物理含义**：多空成交量对比。
- **优势**：在「放量上涨」与「缩量下跌」识别上比 OBV 直观。
- **致命缺陷**：跨周期阈值不稳定。

### 5.3 MFI（资金流量指标）

- **算法**：「价×量」加权的 RSI。
- **物理含义**：带量价权重的超买超卖。
- **优势**：比 RSI 更准（信息量更大），尤其在大资金股上。
- **致命缺陷**：A 股小盘股资金流变化剧烈，短周期 MFI 噪声大。

### 5.4 量比 / vol_ratio_5

- **算法**：`vol_today / MA5(vol)`。
- **物理含义**：今天比近 5 日平均放/缩多少。
- **优势**：**短线突破最重要的确认指标**。突破不放量必假。
- **实战要点**：突破型策略要求 `vol_ratio > 1.5`；反弹型策略要求 `vol_ratio > 1.2` + 阳线收回。

## 6. 组合方法学（理论）

### 6.1 黄金法则三条

1. **跨类别正交化**：组合内指标至少跨 3 个信息源类别（A/B/C/D/E）。
2. **同向确认**：同类别 ≤ 2 个指标，互相验证（如 MACD + ADX 都属趋势加速类）。
3. **趋势权重偏置**：A 类（趋势）应占 30–50%，剩下分给 B/C/D。**纯动量组合在 A 股震荡市必死**。

### 6.2 归一化公式（本仓库标准）

| 指标 | 归一化方法 | 实现 |
|---|---|---|
| RSI、KDJ-K、`boll_pct_b` | `clip(s, 0, 100)` | 直接 |
| WR | `clip(100 + s, 0, 100)` | 反向修正 |
| MACD 柱 / OBV 斜率 / vol_ratio / ADX / -ATR% | `rolling_rank(s, 60) × 100` | 60 日分位 |
| SuperTrend / 多空二元 | `(s + 1) / 2 × 100`（±1）或 `s × 100`（0/1） | — |
| CCI | `clip((CCI + 200)/4, 0, 100)` 或反向 `100 - clip(...)` | 双向 |

### 6.3 信号触发模式

| 模式 | 含义 | 适用场景 |
|---|---|---|
| `cross_down(buy_th)` | 下穿 → 进场 | 反转 / 抄底（如 E2） |
| `cross_up(buy_th)` | 上穿 → 进场 | 突破 / 趋势确认（如 E3） |
| `score > buy_th` 持续 N 日 | 趋势同向锁定 | 趋势跟踪 |
| `signal AND extra_filter` | 软分 + 硬过滤 | 推荐：所有生产策略 |

### 6.4 反模式（禁止）

- ❌ 不归一化直接相加（MACD 量纲爆炸）
- ❌ 同类堆 5 个（RSI + KDJ + StochRSI + WR + ROC = 一件事）
- ❌ 均权 5 因子（无视先验 = 浪费已有知识）
- ❌ 用日内指标决定日级买卖
- ❌ 回测期 < 3 年下结论

## 7. 最优组合方案（本仓库实证最终版）

### 7.1 终极推荐：「**P1+P4 双引擎 + 基本面预筛 + ATR 止损**」

| 组件 | 选择 | 理由 |
|---|---|---|
| **股票池** | `cn_stock_selection` 基本面 Top 89 | V3 实测 PF 提升 30%+ |
| **主策略 P1**（中线波段，主仓） | **S12 硬条件版**（沿用现有实现） | V3 PF **1.81–2.39** 跨所有风险档冠军 |
| **副策略 P4**（卫星仓位，进攻） | **E3 趋势突破** + 全市场池 | V2 PF 2.44，在弹性股上仍有 alpha |
| **风险参数** | `stop = 2·ATR14`, `target = 4·ATR14`, `max_hold = 20d` | 自适应于个股波动 |
| **仓位** | 单笔 ≤ 5% 总资产，主副策略 7:3 | 风控基线 |

### 7.2 推荐复合指标 — `MASTER`（在 V3 之上的最终融合）

我推荐**只入库一个高度精炼的复合指标，作为「全市场默认筛选器」**：

```json
{
  "name": "MASTER A股日线最优筛选器",
  "purpose": "扫描全市场每日，输出『当日值得关注』候选池",

```

**为什么这个组合最优**（基于 V2 + V3 实证）：

1. **跨 4 个信息源类别**（A 趋势 + B 动量 + C 波动 + D 资金），完美正交化。
2. **资金类权重 35%**（OBV 0.20 + 量比 0.15）— V2/V3 都验证 OBV 斜率是 A 股最强 alpha 源。
3. **趋势类权重 30%**（MA 0.15 + MACD 0.15）— 给方向打底但不过度依赖（避免 E1 的「均线趋势已被定价」陷阱）。
4. **波动类权重 25%**（BOLL_pct_b + 缩口）— 同时支持「极值反转」和「波动率突破」两类入场。
5. **动量类权重仅 10%** — 故意压低，避免 T3 / V2 P3-tight 的「同类堆叠陷阱」。
6. **双信号设计**（`buy_strong` 抄底 + `buy_breakout` 突破）— 在同一指标体系下覆盖两种主要市场风格，根据 ADX 动态切换。
7. **ATR 自适应止损** — 比固定 5% 适应性强；对低波动股缩到 2–3%，对高波动股放到 6–8%，跨股票通用。

### 7.3 推荐的「三档生产部署」

| 部署档位 | 股票池 | 主信号 | 风控参数 | 适用用户 |
|---|---|---|---|---|
| **稳健档** | 基本面 Top 89 | `MASTER.buy_strong`（=S12 升级版） | -5/+10/20d | 大多数用户默认 |

| **预警档** | 全市场 | `score < 15` 极端值 + IM 推送 | 不入交易闭环 | 信息推送，不自动下单 |

## 10. 局限与诚实声明

1. 本文档基于 6 年（2020–2025）数据 + 2 个股票池（通用 56 / 基本面 89），不涵盖 2008/2015 极端行情；
2. 基本面 snapshot 是当前值而非 walk-forward 历史值，存在轻微 look-ahead；
3. 没有考虑停牌、ST、退市、新股次新等极端边界；
4. **任何复合指标都是「概率游戏」**，再优秀的组合也有 30–40% 亏损交易。**请配合资金管理（单笔 ≤ 5%、行业分散、最大回撤熔断）使用**；
5. **过去表现不代表未来收益**。建议落地后保留至少 6 个月样本外观察期，胜率/PF 若降低到回测的 60% 以下需要重新调参。
