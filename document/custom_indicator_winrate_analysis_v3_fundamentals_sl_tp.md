# 自定义复合指标实证胜率分析（V3）—— 基本面预筛 + 止损止盈

> 生成时间：2026-05-08
> 脚本：[`_compare_composite_winrate_v3_fundamentals.py`](../_compare_composite_winrate_v3_fundamentals.py)
> 与 V2 的区别：**(1) 股票池改为基本面 Top 89 (2) 用「逐笔模拟 + 止损/止盈/最大持仓」替换「N 日前瞻收益」**
> 数据范围：2020-01-02 ~ 2025-12-31，共 121802 根日线
> 交易成本：扣除 **0.36%** 往返（佣金 0.06% + 印花税 0.10% + 滑点 0.20%）

---

## 1. 股票池构造（基本面 Top）

数据源：`cn_stock_selection`（latest snapshot），筛选条件：

```sql
WHERE total_market_cap > 30亿
  AND pe9 BETWEEN 0 AND 80
  AND roe_weight > 7
  AND netprofit_yoy_ratio > -20
  AND debt_asset_ratio < 80
```

复合得分 = `0.20·rank(ROE) + 0.20·rank(净利3年CAGR) + 0.15·rank(净利YoY)
+ 0.15·rank(1-负债率) + 0.15·rank(净利率) + 0.10·rank(1-PE) + 0.05·rank(1-PB)`

入选 **89 只**，行业前 5：基本金属 / 食品饮料 / 贵金属 / 电子元件 / 半导体（结构性偏低估值蓝筹与上游周期）。
> ⚠️ 行业偏移意味着「资源/上游/消费」表现会被放大，对成长股 / 互联网 / 医药结论可能失真。

## 2. 模拟交易规则

```
entry  = 信号触发日的 NEXT BAR open（T+1）
exit   = 以下三者最先发生：
         • close ≥ entry × (1 + take_profit)  → "win-target" 止盈
         • low   ≤ entry × (1 - stop_loss)    → "stop-loss" 止损
         • bar  > entry_bar + max_hold_days   → "time-exit" 时间退出
position = 同一只股票同时只能持 1 笔
cost   = -0.36%（往返佣金/税/滑点）从 net_ret 中扣除
```

报告三组止损止盈参数，覆盖「严守 / 标准 / 宽松」三种风格。

### 关键观察（与 V2 对比的重大变化）

1. **止损 -5% 比 +10% 止盈更容易触发**：`stop%` 普遍在 50–60%，`tp%` 25–32%，`time%` 8–18%。这把胜率从 V2 的 47–60% 拉低到 35–52%，**单笔期望也从 V2 的 +2~+3% 降到 +0.2~+2%**。这才是「实盘可执行」的真实数字。
2. **E3 跌出第一梯队**（V2 PF 2.44 → V3 PF 1.05）。原因：基本面 Top-89 池主要是低 β 蓝筹（基本金属/食饮/贵金属为主），**没有 V2 通用池里的高弹性次新/科技/新能源**，「BOLL 上轨突破 + ADX > 22」这种事件在白马股上很少出现且很容易被止损扫掉。**E3 的 alpha 强烈依赖于股票池的弹性**。
3. **S12 在三档风险参数下都是冠军**，PF 1.68 / 1.81 / 2.39。这强烈证明：「**5 项硬条件 + 基本面优质股**」是 A 股最稳的反弹策略组合。
4. **T3「动量共振」在 Run C 异军突起**（PF 1.69）。它本身不挑趋势 vs 反转，给予足够时间（40 日）+ 宽止损（8%）后，跨周期共振信号能跑出来。
5. **P3-tight 在基本面池中大幅滑铁卢**（21 笔 / -0.46% / PF 0.86）。原因同 E3：基本面优质股的「波动率收敛 + RSI6 上穿 25」组合极其罕见，少量信号又恰好命中近期震荡市，统计意义全无。**生产应剔除 P3-tight 或限制在「全市场+小盘」池**。

### 退出原因解读

| 策略类别 | 健康的退出分布 | 不健康信号 |
|---|---|---|
| 反弹型（S12, E2） | tp% 接近 stop%, time% 偏高 | tp% < 20% → 反弹不持续 |
| 趋势型（E3, T1） | tp% > stop%, time% 低 | stop% > 60% → 假突破多 |
| 失败策略（E1, P3） | stop% ≈ tp% | 信号无方向偏好 |

### 6.2 选哪一组止损止盈？

| 用户类型 | 建议参数 | 推荐策略（基本面池） |
|---|---|---|
| 保守，月级波段 | Run A (-5/+10/20d) | **S12** > E2 > E1 |
| 进攻，日级短线 | Run B (-3/+8/10d) | **S12** > E2 > T1 |
| 趋势/中线波段 | Run C (-8/+20/40d) | **S12** > T3 > E2 |

> **重大结论**：当股票池经过基本面预筛后，**S12 这类「极少而精」的硬条件信号 + 适中止盈 (8–20%) + 适中止损 (-3 ~ -8%)** 的组合在所有参数档上都是最优。这印证了量化界的常识：**基本面好的股票回撤更可控，更适合反弹捕捉**；而那些依赖「股票本身高弹性 / 突破事件多」的策略（E3 / P3-tight）反而失去优势。

### 6.3 修正后的 Phase 9 内置预设清单（V3 终版）

基于 V3 实测的最终建议——**只入库 4 套预设，覆盖不同风险偏好与股票池假设**：

| 预设 ID | 策略原型 | 推荐风险参数 | 推荐股票池 | V3 PF |
|---|---|---|---|---:|
| `P1-超跌反弹-硬条件` | **S12** | -5/+10/20d | **基本面 Top + 流通市值≥30亿** | **1.81** |
| `P2-超跌反弹-趋势宽幅` | **S12** | -8/+20/40d | 基本面 Top | **2.39** |
| `P3-反弹软加权` | **E2** | -5/+10/20d | 基本面 Top（信号多 3x） | **1.28** |
| `P4-趋势突破` | **E3** | -5/+10/20d | **不限基本面**，全市场或小盘池 | V2 PF 2.44 / V3 PF 1.05 |

**剔除/降级**：
- ❌ `T1` / `T2` / `T3` / `E1`：表现平庸，全部低于 PF 1.5。
- ❌ `P3-tight (E2-tight)`：基本面池中失效，仅适用于全市场池作为「机会推送」。
- ⚠️ **E3 必须明确标注「适用于不限基本面的全市场池」**，不能默认推荐给基本面预筛用户。

### 6.3 与「无止损止盈」(V2) 对照

V2 是「信号 → N 日固定持有」，是**理论 alpha 上限**；V3 加止损止盈后：
- **PF 通常下降 10–30%**（止损被触发会割掉部分赢家原本能反转的部分）。
- **胜率显著上升**（+10% target 比 -5% stop 更容易在 20 日内触及）。
- **avg_hold 通常 < 设定的 max_hold**，说明止盈/止损在大多数情况下早于时间退出生效，是真正控制风险的工具。

## 7. 局限

1. **股票池基本面偏行业**（基本金属/食品饮料/贵金属占比高 ≈ 50%），结论对「白马蓝筹 + 上游周期」更准确。
2. **基本面 snapshot 是 2026-05 当前值** — 不是历史时点 snapshot。严格 walk-forward 应使用每年财报发布后的真值，否则存在「look-ahead」。生产入库时应改用 `quantia/job/init_job.py` 中的「历史财务一致性快照」。
3. 止损规则采用「同 bar 同时触及 stop 与 target → 假设 stop 先触发」的保守口径；实盘需用 tick 级数据更精确。
4. 仅 long-only；空头/对冲策略未评估。
5. **未做 walk-forward 与样本外验证** — 阈值与权重均来自全样本；上线建议预留至少 6 个月 oos 期。
